AONX Server
-----------

Pequeño servidor de Argentum Online.

------------------

Este software se encuentra publicado bajo la licencia GNU GPL v3. El uso del 
software implica aceptar las condiciones mencionadas en la licencia.

Para obtener más detalles sobre las condiciones de uso puede leer la licencia
de uso completa, ubicada dentro del archivo licence.txt. 

Instrucciones
-------------

Para ejecutar el servidor hace falta tener instalado Python y Twisted. En 
Debian y Ubuntu eso se logra con:

    aptitude install python-twisted

Una vez instalados, el servidor se inicia corriendo el script "runserver.py".

    python runserver.py
